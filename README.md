## 📖 Introduction | 简介

**DailyClaw** is a **continuously evolving** AI Agent project that documents the entire journey from initial ideas to a complete multi-agent collaboration system.

> 🎯 **Goal**: Build an observable, reproducible, and learnable AI Agent evolution milestone
>
> 📈 **Philosophy**: Small daily improvements, witnessed through version history

**DailyClaw** 是一个**持续进化**的 AI Agent 项目，记录了从最初的想法到完整多代理协作系统的全过程。

> 🎯 **目标**：打造可观测、可复现、可学习的 AI Agent 进化里程碑
>
> 📈 **理念**：每天进步一点点，用版本历史见证成长

## 📚 Usage | 使用

### Daily Evolution | 每日进化

The project automatically records daily progress at 00:00 via GitHub Actions:

```yaml
# .github/workflows/daily-evolution.yml
name: Daily Evolution
on:
  schedule:
    - cron: '0 0 * * *'  # Daily at 00:00 UTC
```

### Manual Trigger | 手动触发

```bash
# Create today's evolution record
./scripts/create-evolution.sh

# Update progress dashboard
./scripts/update-progress.sh
```
